# grid
css responsive grid

##viewports##
- mobile: smaller 500px
- tablet: bigger 500px
- desktop: bigger 768px

##usage##

css class usage:

###floats###
- fr: float right
- fl: float left
- cf: clear float

###rows###

- r: row

###columns###
nesting possible. column widths change on smaller viewports
- c1: 100%
- c1_2: 50%
- c1_3: 33.33%
- c2_3: 66.66%
- c1_4: 25%
- c3_4: 75%
